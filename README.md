# Diplomski rad 

Ovo je repozitorijum mog diplomskog rada iz oblasti računarstva i automatike, izrađenog na Fakultetu tehničkih nauka, Novi Sad.

## Tehnologije
- Python
- Angular
- HTML / CSS / JavaScript

## Opis projekta
- Backend (Python) – obrada podataka, skripte, API
- Frontend (Angular) – korisnički interfejs
